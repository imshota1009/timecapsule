# 📘 LEARN.md

## 🧭 Overview

**Time Capsule** is a web app that lets you write a letter to your future self and have it delivered by email on a date you choose.

This project is also designed as an educational resource for learning **modern web development with Firebase**.

---

## 🔍 What You Can Learn

By exploring the code, you can learn:

1. 🌐 **Single-Page App Architecture**
   - Page-based navigation without a framework
   - DOM manipulation and dynamic content rendering

2. 🔥 **Firebase Integration**
   - Firestore for real-time cloud database storage
   - Cloud Functions for scheduled background tasks
   - Firebase Hosting for deployment

3. 🌍 **Internationalization (i18n)**
   - Building a bilingual app (English / Japanese)
   - Language auto-detection from browser settings
   - Dynamic text switching with translation dictionaries

4. ✨ **CSS Animations & Visual Design**
   - Starfield background with CSS animations
   - Glassmorphism and gradient effects
   - Smooth page transitions and micro-interactions

5. 📧 **Email Automation**
   - Scheduled Cloud Functions for daily email delivery
   - HTML email template design
   - Nodemailer integration with Gmail

6. 📱 **Responsive Design**
   - Mobile-first layout using CSS custom properties
   - Fluid typography and adaptive spacing

---

## 🛠️ Key Technologies

- **HTML5** — App structure and semantic markup
- **CSS3** — Custom properties, animations, responsive design
- **JavaScript (ES6+)** — App logic, i18n system, Firebase SDK
- **Firebase Firestore** — Cloud database for capsule data
- **Firebase Cloud Functions** — Scheduled email delivery
- **Firebase Hosting** — Deployment and CDN
- **Nodemailer** — Email sending via Gmail

---

## 🧪 Try It Out

1. Clone the repository
2. Open `public/index.html` in your browser for a local preview
3. Write a letter, choose a delivery date, and seal the capsule
4. Explore the code to see how Firebase and the i18n system work
5. Modify the translation dictionary to add another language!
